8051-stc8a8k64d4
	基于 STC8A8K64D4 的 KEIL C51 项目。
	调试串口是串口1，使用定时器1作为波特率发生器；定时器0 用于串口1 RXD 的超时检测。
	该项目虽然开启了时间信息输出，但是并没有实现更新时间的语句，因此时间输出时钟为 [00:00:00]

windows
	可以通过 vscode 的 run code 插件进行编译和运行。
	
调试时，需要采用支持 VT100 控制码的控制台，否则输出打印时会异常。

指令简要说明：
help：打印所有命令的名称和说明
ver：打印版本号
vis：设置回显是否显示
clear：控制台清屏
user：回显用户输入，同时回显输入的参数
quit：退出 mysh 控制台
